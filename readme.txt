***Java Joystick to Key ***

To run:
    On windows: double click on JavaJoystickToKey.exe
    On linux: double click on the .sh that corresponds to your architecture, or cd to the directory of the file and run 'sh JavaJoystickToKey-Linux**.sh'

This application uses the Java Joystick library, which can be found at:

http://sourceforge.net/projects/javajoystick/
